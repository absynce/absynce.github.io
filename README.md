# absynce.github.io
absynce developer blog - why I do the things I do on the slippery path to mastery. Written in Elm and English.

# Getting started

## Install dependencies

Install this:
* [Node.js](https://nodejs.org/)

Then run the following at the command line:
```
npm install
```

## Start coding

```
npm run dev
```

## Other commands

```
npm run build
npm run start
npm run test
npm run watch
```
